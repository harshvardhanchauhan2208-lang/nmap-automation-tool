This project is a simple network scanning tool built using Python and Nmap.
It allows users to scan a target IP address and discover open ports and running services.

The tool provides multiple scan options such as SYN scan, UDP scan, and a comprehensive scan for deeper analysis.

It helps beginners understand how network scanning works in cybersecurity.

Features:
Scan target IP addresses
Detect open ports
Identify protocols
Perform SYN scan
Perform UDP scan
Run advanced Nmap scans
Display scan information and host status

Technologies Used:
Python
Nmap
python-nmap library
